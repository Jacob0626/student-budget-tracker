# student-budget-tracker
### A Streamlit app for tracking student income, expenses, budgets, and savings goals.
---

This project will be a Streamlit app for tracking student income, expenses, budgets, and savings goals.

The main idea is that a user will be able to upload a CSV file with income and expense data. The app will organize the data into categories and show summaries such as total income, total expenses, money left over, spending by category, and monthly spending trends.

I also want to add features like:
- editing or adding transactions inside the app
- filtering expenses by date, category, and type
- setting monthly budgets for categories like food, gas, subscriptions, and entertainment
- setting a savings goal
- saving the user’s budget settings
- downloading or exporting a summary report
